# Prevent Glass / Print Exposure Guide

For repository safety:

1. Keep repository private.
2. Limit collaborator roles to minimum required access.
3. Use CODEOWNERS + branch protection for all source/config paths.
4. Never store credentials in docs or tracked files.
5. Serve API docs through static docs files without secrets.

For local environments:

- Avoid printing sensitive data to terminal logs.
- Keep screenshots/screen shares free of tokens and `.env` values.
